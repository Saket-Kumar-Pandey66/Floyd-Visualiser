Algorithm Info

How It Works

Floyd's algorithm uses two pointers moving at different speeds. Slow pointer moves 1 step, fast pointer moves 2 steps at a time.

Detection Logic

If no cycle exists, fast pointer reaches the end. If a cycle exists, the fast pointer catches up to the slow pointer.

Complexity

Time: O(n)

Space: O(1)

struct Node* detectCycle(struct Node* head) {
    struct Node* slow = head;
    struct Node* fast = head;
    
    while (fast != NULL && fast->next != NULL) {
        slow = slow->next;
        fast = fast->next->next;
        
        if (slow == fast) break;
    }
    
    if (fast == NULL || fast->next == NULL) 
        return NULL;
    
    slow = head;
    while (slow != fast) {
        slow = slow->next;
        fast = fast->next;
    }
    
    return slow;
}
